# React Fibers & Server Components POC

A Proof of Concept project exploring React Fibers architecture, React Server Components, and modern CSS features.

## What's Inside

- **React Fibers**: Examples demonstrating concurrent rendering and work scheduling
- **Server Components**: Server-side rendering and streaming demonstrations
- **Modern CSS**: Interactive examples of CSS Grid Subgrid, Container Queries, Cascade Layers, and more

## Tech Stack

- Next.js 14+
- React Server Components
- TypeScript
- Tailwind CSS
- SCSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
